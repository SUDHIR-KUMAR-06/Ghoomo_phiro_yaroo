# Frontend

React + Vite frontend for the HIMTREK 2026 landing page.

## Commands
- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Structure
- `src/components/` - Reusable UI sections and shared view components
- `src/pages/` - Page-level React components
- `src/assets/` - Placeholder location for future images and media
- `src/styles/` - Centralized CSS files
