# HIMTREK 2026 Workspace

Full-stack monorepo scaffold for a premium trekking landing page and a future backend.

## Structure
- `frontend/` - React + Vite app with reusable components, central styles, and placeholder image zones
- `backend/` - Minimal backend scaffold for future API work
- `project.code-workspace` - VS Code workspace file

## Run
1. Open the repository in VS Code.
2. In `frontend/`, run `npm install`.
3. In `frontend/`, run `npm run dev`.

## Notes
- All image zones are styled placeholders named `image1`, `image2`, `image3`, and `image4`.
- The frontend is organized for direct editing in VS Code with modular React components.
